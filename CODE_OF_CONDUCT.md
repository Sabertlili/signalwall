# Code of Conduct

SignalWall follows the Contributor Covenant Code of Conduct, version 2.1.

## Our pledge

We pledge to make participation in this project respectful, focused, and welcoming for everyone.

## Expected behavior

- Be direct and constructive.
- Assume good intent, but fix unclear or harmful behavior quickly.
- Keep feedback tied to the project, code, docs, design, or user experience.
- Respect different experience levels and different Windows setups.

## Unacceptable behavior

- Harassment, insults, threats, or personal attacks.
- Publishing private information without permission.
- Repeated off-topic disruption.
- Security reports posted publicly before the maintainer has a chance to respond.

## Enforcement

The maintainer may remove comments, close issues, block users, or limit participation when needed to protect the project.

For the full Contributor Covenant text, see https://www.contributor-covenant.org/version/2/1/code_of_conduct/.
