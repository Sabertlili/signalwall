# Changelog

All notable changes to SignalWall will be documented here.

## v0.3.0

- Added import and export for shareable SignalWall preset files.
- Added strict preset normalization and host-side configuration validation.
- Added a safe screen-order fallback when the configuration file is damaged.
- Added automated tests for configuration persistence and monitor-slot mapping.
- Added SPDX SBOM generation, GitHub build provenance, and SBOM attestations.
- Added a machine-readable release manifest and local verification script.
- Pinned the WebView2 dependency and updated GitHub Actions to current stable releases.

## v0.2.0

- Added a built-in WebView2 control center.
- Added screen ordering, global/per-screen modes, transition selection, and quote/theme editors.
- Added light and dark control-center themes.
- Added a branded application icon and expanded tray menu.
- Added tag-driven installer and portable-package releases with SHA-256 checksums.
- Refreshed the website and Product Hunt media with real product captures.
- Updated GitHub Actions dependencies.

## v0.1.1

- Added open-source community files, issue templates, CI, CodeQL, Dependabot, and launch documentation.
- Added clearer source-first installation language.
- Added bilingual README and AI-assisted install prompts.

## v0.1.0

- Initial alpha release.
- Multi-monitor WebView2 wallpaper app.
- Bundled Quote Signal wallpaper.
- Tray controls for reload, web folder, and exit.
- Basic installer build workflow.
