# Support

SignalWall is an early open-source project. Support is best effort.

## Before opening an issue

1. Read the [AI-assisted source install prompt](docs/ai-assisted-install.en.md).
2. Confirm your Windows version.
3. Confirm whether you ran from source or used a release binary.
4. Run:

```powershell
dotnet build .\src\SignalWall\SignalWall.csproj -c Release
```

## Where to ask

- Bugs: open a bug report issue.
- Ideas: open a feature request issue.
- Security: use the private process in [SECURITY.md](SECURITY.md).

## Useful details

Include monitor count, scaling percentage, Windows version, WebView2 status, and screenshots when the issue is visual.
