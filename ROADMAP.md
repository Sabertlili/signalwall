# Roadmap

SignalWall is intentionally early. This roadmap keeps the project direction clear without pretending the alpha is finished.

## Now

- Source-first install flow.
- Multi-monitor wallpaper windows.
- Quote Signal as the bundled default wallpaper.
- Per-screen theme and quote behavior.
- Clear README, screenshots, and security posture.

## Next

- Signed installer path or clearer release verification artifacts.
- Built-in settings window for non-technical users.
- Import/export for quote packs and visual themes.
- Startup toggle.
- Better pause behavior for fullscreen apps and games.
- Automated smoke tests for wallpaper startup.

## Later

- Wallpaper library.
- Community theme gallery.
- Optional auto-update flow with explicit documentation.
- More wallpaper engines beyond HTML/CSS/JS.

## Non-goals for now

- Copying Lively Wallpaper source code.
- Hidden telemetry.
- Background persistence that is not user-visible.
- Disabling Windows security to make unsigned binaries easier to run.
